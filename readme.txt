Простейший пример приложения на django с исп. функций в качестве представлений
Пример работы с poetry в качестве пакетного менеджера

API:
GET| http://127.0.0.1:8000/vacancies/ - список всех вакансий из БД
GET| http://127.0.0.1:8000/vacancies/?text=test vack #232 - отдаст вакансию с данным описанием
GET| http://127.0.0.1:8000/hello/ - прост тест
GET| http://127.0.0.1:8000 - index_page
POST http://127.0.0.1:8000/vacancies/ - добавление вакансии
GET  http://127.0.0.1:8000/admin/ - панель админа с добавленной моделью Вакансии

Приложение написал в качестве ознакомления с Django с использованием CBV
